# Predicting Student Re-engagement with AI Assistants Using Supervised Machine Learning
## Description
This project predicts whether a student is likely to use an AI assistant again based on their AI usage behavior, academic details, and satisfaction level using a Random Forest Classifier.

## Dataset
* Dataset: AI Assistant Usage in Student Life
* Total Records: 10,000
* Target Variable: UsedAgain

## Project Files
* **data/** – Dataset used for training and testing.
* **model/** – Trained Random Forest model (.pkl).
* **notebook/** – Jupyter Notebook containing the complete implementation.
* **report/** – Project report in PDF format.

## Machine Learning Model
* Random Forest Classifier

* Scikit-learn
